是我在小组项目 CPT202 软件工程中sprint 4， 5所完成的项目 
但是因为与寻物购物商城的兼容性，以及时间问题没有完成 
所以单独做成一个项目 
仅供学习使用，沙箱支付秘钥属于HAITAO.BU，除CPT202项目老师检查外请勿外传，谢谢
